# Inventaire de matériel informatique

C'est un projet JavaFx fxml pour la gestion d'inventaire de matériel informatique.

## Fonctionnalités

### Fonctionnalité Formule ✨

La fonctionnalité **Formule** permet d'effectuer des calculs automatiques pour la gestion de l'inventaire :

- **Calculs de coûts** : Coût total, coût par utilisateur, TCO (Total Cost of Ownership)
- **Amortissement** : Calculs linéaires et dégressifs
- **Valorisation** : Valeur résiduelle, taux de dépréciation
- **Analyse d'inventaire** : Taux de renouvellement, âge moyen du parc, valeur totale
- **Formules personnalisées** : Créez vos propres formules adaptées à vos besoins
